<i>Trollslate</i> transforme votre texte avec un style de code barre. Amusez-vous à faire deviner à vos amis qu'est-ce qu'il est écrit. N'écrivez pas trop d'infamie, ça n'a pas d'intérêt. Pensez à augmenter la luminosité de votre écran si vous ne voyez rien. Trollslate fonctionne mieux sur certains écrans mais le votre devrait faire l'affaire quand même.
